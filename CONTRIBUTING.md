# Contributing to Caffeinator

Thanks for your interest in Caffeinator! This project is intentionally small, focused, and maintained by a single developer. To keep the codebase simple and consistent, **pull requests are not accepted**.

That said, your feedback is welcome and appreciated.

## How to contribute

### 🐞 Bug Reports
If you’ve found a bug, please open an issue using the **Bug Report** template.  
Include steps to reproduce, expected behavior, and your macOS version.

### 💡 Feature Requests
If you have an idea that would make Caffeinator more useful, feel free to open a **Feature Request** issue.  
Not all requests will be implemented, but all will be read.

### 📫 Questions
If you’re unsure whether something is a bug or expected behavior, open an issue and I’ll take a look.

## Pull Requests
Caffeinator does **not** accept pull requests.  
This helps keep the project’s scope tight and ensures the codebase remains consistent and maintainable.

If you’d like to propose a change, please open an issue instead.

## Thank You
Your interest and feedback help make Caffeinator better.  
Thanks for taking the time to contribute!
